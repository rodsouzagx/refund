### About the Project

**Refund** is a web application designed to manage expense reimbursement requests. Its goal is to allow employees or users to log their expenses (such as food, transport, accommodation, etc.) in a simple and organized way.

This project serves as a practical example of DOM manipulation and web form structuring.

### Technologies Used

- **HTML5**: Used for the semantic structure of the page. The code utilizes modern tags like `<main>`, `<section>`, `<fieldset>`, and `<aside>` to ensure accessibility and good organization.
- **CSS3**: Responsible for the visual styling of the application (`styles.css` file).
- **JavaScript**: Responsible for interactivity, such as capturing form submissions, visually adding items to the list, and calculating totals (`scripts.js` file).

### Features

- **Expense Registration**: A form to input the expense name, select a category, and enter the monetary amount.
- **Categorization**: Use of icons and predefined categories (Food, Accommodation, Services, Transport, Others).
- **Financial Summary**: A sidebar (`aside`) that displays the count of logged expenses and the total accumulated value.

### How to Run

1.  Clone or download this repository.
2.  Ensure that `styles.css`, `scripts.js`, and the `img` folder are in the same directory as `index.html`.
3.  Open the `index.html` file in your preferred web browser.
